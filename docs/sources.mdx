# Sources

Sources connect external data/tools to Normies.

Supported source types:
- MCP servers
- REST APIs
- Local folders

## What to Configure

- `config.json` for connection/auth details
- `permissions.json` for safe defaults in Explore mode
- `guide.md` with context-specific usage notes

## Validation

After creating a source, run source validation (`source_test`) before considering setup complete.
