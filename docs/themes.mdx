# Themes

Themes control the visual style of Normies.

You can:
- Use built-in themes
- Override theme tokens
- Keep a consistent brand look for your team

## Theme Scope

Themes can be applied at:

- App default level
- Workspace override level

This lets teams share a consistent baseline while allowing exceptions where needed.

## Practical Guidance

- Start with a preset, then make small token adjustments.
- Verify readability in both light and dark contexts.
- Keep contrast high for long-running task sessions.
